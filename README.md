# Organizador de Biblioteca

## Descripción

El Organizador de Biblioteca es una aplicación desarrollada en C++ que permite administrar el registro de libros de una biblioteca mediante un menú interactivo. El sistema facilita la gestión del inventario, el control de préstamos y devoluciones, y la consulta de la información de los libros.

## Objetivo

Desarrollar un sistema que permita gestionar de forma organizada los libros de una biblioteca, aplicando estructuras de datos, funciones y trabajo colaborativo con Git y GitHub.

## Funcionalidades

- Registrar libros.
- Mostrar libros registrados.
- Buscar libros.
- Modificar información de un libro.
- Eliminar libros.
- Ordenar libros.
- Registrar préstamos.
- Registrar devoluciones.
- Generar reportes de libros.

## Tecnologías utilizadas

- Lenguaje: C++
- Biblioteca: rlutil
- Control de versiones: Git
- Repositorio remoto: GitHub
